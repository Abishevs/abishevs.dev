# Project

The site already contains:

- Posts
- Projects
- Project Journal
- About

Future content types may be added.

The redesign should enhance presentation while preserving compatibility with existing content.

The website is not a traditional portfolio, startup landing page,
or corporate CV.

It is a public engineering notebook containing:

- Project documentation
- Project journals
- Learning notes
- Technical blog posts
- Books
- Movies
- References
- Personal tooling and workflows

The primary audience is:

- Curious engineers
- Embedded developers
- Makers
- Linux users
- Technical students
- Future collaborators

The website should feel like:

- An engineering notebook
- A technical field journal
- A systems exploration log

The website should NOT feel like:

- A SaaS landing page
- A startup website
- A flashy portfolio
- Cyberpunk UI
- Gamer aesthetic

Core themes:

- Curiosity
- Exploration
- Systems thinking
- Experimentation
- Documentation
- Engineering

# Design System

## Theme Mode

Dark mode is the primary and only supported mode.

Do not implement light/dark switching.

Design for long reading sessions.

Prioritize low eye strain.

Avoid pure black backgrounds.

## Visual Inspiration

Primary influences:

- Modern technical documentation
- Engineering notebooks
- Technical manuals
- Instrumentation displays
- PCB design aesthetics
- Research journals

Secondary influences:

- Project Hail Mary
- The Martian
- NASA mission reports

These influences should be subtle.

Do not create literal sci-fi interfaces.

Do not create spaceship dashboards.

## Typography

Preferred fonts:

Headings:
- IBM Plex Sans

Body:
- Source Serif 4
- IBM Plex Serif

Code:
- JetBrains Mono

Typography should prioritize readability.

## Colors

The site should use calm, technical colors.

The visual language should resemble:

- Terminal applications
- Technical instrumentation
- Engineering software
- Dark code editors

Primary palette:

Background:
#111827

Surface:
#1F2937

Text:
#E5E7EB

Muted Text:
#9CA3AF

Telemetry Blue:
#38BDF8

Success:
#22C55E

Warning:
#F59E0B

Error:
#EF4444

Avoid:

- Pure black backgrounds
- Neon colors
- RGB aesthetics
- Cyberpunk palettes
- High saturation accents

## Design Goals

Prioritize:

- Reading experience
- Information density
- Clarity
- Structure

Do not prioritize:

- Animations
- Visual effects
- Decorative graphics

# Project Cards

Projects are a first-class content type.

Project cards should display:

- title
- description
- status
- projectType

Optionally:

- thumbnail

Status should be visually emphasized.

Project type should be displayed as metadata.
